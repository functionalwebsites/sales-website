Frequently Asked Questions

What happens when I upgrade to Pro? ▼
Your free projects carry over. You immediately unlock direct deployment, marketplace access, custom blocks, custom templates, and the rest of the Pro builder features. No setup needed.
Can I export my site and host it elsewhere? ▼
Yes. Export your site as a ZIP file containing all HTML, CSS, and JavaScript. You can host it on Netlify, GitHub Pages, AWS S3, or anywhere static files are served.
Do I need technical skills? ▼
No. The builder is designed for non-technical users. Drag, drop, and publish. For advanced users, you can edit the HTML directly.
What's the difference between exporting and deploying? ▼
Export downloads your site as files and stays free. Deploy publishes your site directly from the builder to the web with a custom domain and HTTPS, and that direct deployment feature requires Pro.
Is there a storage limit? ▼
Your project is stored in local storage on your device, so there's no fixed limit. However, very large projects may impact performance. We recommend keeping individual project sizes under 100MB for optimal experience. Because of this, we can NOT recover your projects if you lose them, make sure to deploy to github often or download the project files as a backup!
What about SEO? ▼
Pro plans and above include SEO tools: meta tag editors, automatic sitemaps, Open Graph tags, and structured data generation. Static sites also load faster, which helps ranking.
What happens if I lose my project? ▼
Because projects are stored in your browser's local storage, we cannot recover lost projects. To prevent loss, we recommend regularly exporting your project files as a backup or deploying to GitHub Pages for safekeeping.
