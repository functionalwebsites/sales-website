Static is not best for everything.

If you need logged-in user accounts, live dashboards, marketplaces, or constantly changing app data, static takes more planning. Complex static builds, including e-commerce, are still an option; they just require more integrations. For a business-card website, portfolio, service site, landing page, docs site, or simple marketing site, static is usually a no-brainer.
