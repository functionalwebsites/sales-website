Custom Blocks

Extend your site builder with reusable, modular components. Create once, use everywhere.

What Are Custom Blocks?

Custom blocks are reusable components you create or import to extend the site builder's functionality. Instead of rebuilding the same component on multiple pages, create it once as a custom block and reuse it with a single click.

Examples of custom blocks include:

  • Testimonial sections with ratings
  • Pricing tables
  • Product cards
  • Contact forms with validation
  • Video embeds (YouTube, Vimeo, etc.)
  • Interactive galleries
  • Call-to-action sections

Using Custom Blocks

Finding Blocks

  1. Click the "Blocks" button in the toolbar
  2. Scroll to "Custom Blocks" section
  3. Browse available blocks or search by name

Adding a Block to Your Page

  1. Drag the block from the library onto your canvas
  2. The block appears on your page with default content
  3. Click the block to select it and edit in the properties panel
  4. Customize text, colors, images, and settings

Configurable Options

Each custom block has specific properties you can edit:

  • Content (text, images, links)
  • Colors (background, text, accents)
  • Layout (spacing, alignment, sizing)
  • Interactivity (hover effects, animations)
  • Block-specific settings (e.g., YouTube URL for video embed)

Creating Custom Blocks

⭐ Pro Feature: Creating and uploading custom blocks is available in the Pro plan. Free users can use existing blocks but cannot create new ones.

Create a Custom Block

  1. Design your component on a page (section, content, styling)
  2. Select the entire component
  3. Right-click and choose "Convert to Custom Block"
  4. Give your block a name, description, and thumbnail
  5. Click "Save as Block"

Your Block is Now Reusable

Your custom block appears in the blocks library under "My Blocks". Drag it onto any page to reuse it. Edit each instance independently—changes to one don't affect others (they're instances, not linked).

Uploading Blocks to the Marketplace

Pro users can share their custom blocks with the community via the Marketplace:

  1. Go to Account > My Blocks
  2. Click the block and select "Publish to Marketplace"
  3. Add a description, preview images, and pricing (free or paid)
  4. Submit for review

Once approved, your block is available for other users to download and use. You can earn revenue from paid blocks!

Block Structure (For Advanced Users)

Custom blocks are stored as JSON files containing:

  • Structure: HTML markup
  • Styling: CSS
  • Interactivity: JavaScript (optional)
  • Metadata: Name, description, preview image
  • Schema: Configurable properties

Advanced users can edit the JSON directly if needed. Find your blocks in the project folder under /site-builder/blocks/.

Best Practices

Design for Reusability

When creating a custom block, think about how you'll use it on multiple pages. Make it flexible—allow text, color, and image customization.

Keep It Modular

Break complex designs into smaller, focused blocks. A "Card" block is more reusable than a "Hero + Card + CTA" block.

Document Your Block

Add a clear description and preview image so you remember what the block does.

Test Responsiveness

Make sure your custom blocks look great on mobile, tablet, and desktop.

Optimize for Performance

Avoid heavy JavaScript or large images in blocks. Keep them lightweight for fast loading.

Marketplace Discovery

Browse community-created blocks in the Marketplace:

  • Visit /marketplace
  • Filter by category or search for what you need
  • Preview blocks with screenshots and demos
  • Click "Add to Project" to install

Troubleshooting

Block Not Appearing?

Make sure it's published and enabled. Go to Account > My Blocks and check the toggle.

Block Lost When Exporting?

All custom blocks are included when you export as HTML. They're embedded in the exported files.

Need to Edit a Block?

Locate the block in the blocks library, select it, and use the properties panel. Or recreate it by converting a new component.

💡 Pro Tip: Save time by creating blocks for your most-used layouts. A testimonial block, pricing table, and CTA section can halve your design time!
